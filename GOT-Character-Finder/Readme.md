# ⚔️ GOT Character Finder

A Flask-based web application that recommends the most similar **Game of Thrones** characters using Machine Learning and Natural Language Processing.

## 🚀 Features

- 🔍 Search any Game of Thrones character
- 🤖 Get similar character recommendations
- 📚 Machine Learning based similarity engine
- 🌐 Interactive web interface using Flask
- ⚡ Fast recommendation using precomputed similarity matrix

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/mohitkhangar/GOT-Character-Finder.git
```

Go to the project directory

```bash
cd GOT-Character-Finder
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000/
```
